## Instructions to Download Images:

1. Visit each product page URL above
2. On each product page, you should see:
   - Package image (the box/package)
   - Glove image (the actual glove)
   - Texture image (close-up texture detail)
3. Right-click on each image and "Save image as..."
4. Save them to: `public/images/gants/` folder
5. Name them exactly as shown in the code:
   - `[Product Name] Package.jpg`
   - `[Product Name].jpg` (for glove)
   - `[Product Name] Texture.jpg`

## Example Naming:
- `Amazing Nitrile Package.jpg`
- `Amazing Nitrile.jpg`
- `Amazing Nitrile Texture.jpg`
